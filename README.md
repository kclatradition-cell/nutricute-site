# NutriCute – Site vitrine compatible Vercel / GitHub / Shopify / Odoo

Site Next.js prêt pour déploiement Vercel.

## Installation locale
```bash
npm install
npm run dev
```

## Déploiement
1. Créer un dépôt GitHub.
2. Envoyer ce dossier dans le dépôt.
3. Importer le dépôt dans Vercel.
4. Ajouter les variables d’environnement si besoin.

## Compatibilité Shopify / Odoo
- Les boutons `Acheter`, `Découvrir`, `Commander` peuvent pointer vers Shopify, Odoo eCommerce ou un formulaire.
- Les produits sont gérés dans `app/page.jsx`.
- La vidéo peut être remplacée par `/public/nutricute-mode-utilisation.mp4`.
